# 字符串拼接

虽然我们可以使用 `+` 运算符来拼接字符串，

但为了效率考虑，

拼接大量字符串时应使用 join 方法：


<v-click>

有时用拼接的方式从片段打印会显得笨拙

</v-click>


::right::

```py {monaco-run} {height:'120px'}
a = "first"
b = "second"

print(a + b)
print(" ".join([a, b, b, a]))   # 稍后会更详细地介绍 join 方法

```

<v-click>

```py

print(str(1) + " plus " + str(3) + " is equal to " + str(4))

print(1, "plus", 3, "is equal to", 4) # 稍微好一些

```

</v-click>
