字符串是 <span v-mark.underline.red="1"> 字符的序列 </span>，

通常用于 <span v-mark.underline.red="1"> 在程序中存储输入或输出数据。 </span>

<v-click>

字符串的字符可以用单引号 `'` 或双引号 `"` 括起来。

如果字符串需要包含引号，这会很有用：`"I don't want to go!"`。

</v-click>

<v-click>

你也可以通过在引号前加反斜杠来转义它：`'I don\'t want to go'.`

</v-click>

<v-click>

字符串还可以包含其他转义序列，例如 `\n` 表示换行，`\t` 表示制表符。

</v-click>

<v-click>

包含换行的字符串可以轻松地用三个双引号或三个单引号括起来

</v-click>

::right::

```py

print("One\tTwo\nThree\tFour")

```

<v-click>

```py

s = """A string
spanning over
several lines"""

print(s)

```

</v-click>
