# 那么，什么是字符串？

<v-click>

<span v-mark.box.red="1"> **引号** </span> 用于表示字符串。

</v-click>

<v-click>

字符串是 <span v-mark.highlight.yellow="2"> 字符的序列。 </span>

它们可以由 <span v-mark.highlight.yellow="3"> 字母、数字 </span> 和任何其他类型的字符组成，例如 <span v-mark.highlight.yellow="3"> 标点符号。 </span>

字符串 <span v-mark.highlight.red="4"> 不仅仅是单词 </span>，单个字符串可以长达多个完整的句子。

字符串通常会 <span v-mark.highlight.red="5"> 完全按照书写的方式打印出来。 </span>

因此，这两个命令会产生两个截然不同的结果

Python <span v-mark.underline.red="6"> 不会计算 </span> 运算的结果，而是 <span v-mark.underline.red="6">将运算本身</span> 作为字符串打印出来。

</v-click>

::right::

```py {monaco-run} {height:'150px'}

print(2 + 2 * 10)
print("2 + 2 * 10")

```
